Common HTML Tags and Their Functions

  1. <p>: Defines a paragraph of text.
  2. <a>: Creates a hyperlink to another webpage.
  3. <h1> - <h6>: Define headings, with <h1> being the highest level and <h6> the lowest.
  4. <img>: Embeds an image in the webpage.
  5. <div>: Creates a block-level container for other HTML elements.
